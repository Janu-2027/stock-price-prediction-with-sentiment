# Stock Price Prediction App

## How to Run Locally
1. Clone this repo:
```bash
git clone https://github.com/your-username/my-streamlit-app.git

Navigate to the project folder:
cd my-streamlit-app

Install dependencies:
pip install -r requirements.txt

Run the app:
streamlit run app.py
